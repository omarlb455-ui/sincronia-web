# Sincronia Web

Landing oficial de Sincronia — atención al cliente con IA para Shopify.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Páginas
- `/` — Landing
- `/privacidad` — Aviso de privacidad LFPDPPP
- `/terminos` — Términos y condiciones
